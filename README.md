# Salesforce SOQL Query App

## Overview
The Salesforce SOQL Query App allows users to query and manage Salesforce data easily using SOQL, Salesforce's Object Query Language.

## Features
- User-friendly interface
- Supports basic and advanced SOQL queries
- Ability to manage query results efficiently
- Export query results in various formats (CSV, JSON)
- Dynamic querying based on user input

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/ChrisSleator/sqapp.git
   ```
2. Navigate to the project directory:
   ```bash
   cd sqapp
   ```
3. Install the required dependencies (if applicable):
   ```bash
   npm install
   ```

## Setup Instructions
1. Ensure you have the necessary API access to your Salesforce account.
2. Configure your Salesforce credentials in the configuration file or environment variables.
3. Run the application:
   ```bash
   npm start
   ```

## Usage
- Enter your SOQL query in the provided input field.
- Click the "Execute Query" button.
- View and manage the query results displayed on the screen.
- Use the export feature to save results in your preferred format.
